# Flight Noise Map

Web app that shows live aircraft over your region using OpenSky Network data.

## Features
- Live map
- Aircraft type, altitude, speed, distance
- Place search with suggestions (Nominatim, Photon fallback)
- Radius slider to limit aircraft range
- Ground traffic toggle (shows in different color)
- Noise score from 1 to 10
- Current location marker on the map
- Aircraft markers oriented by heading

## Development
- Open `index.html` in a browser.

## Data source
OpenSky Network API
