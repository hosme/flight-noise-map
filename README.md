# Flight Noise Map

Web app that shows live aircraft over your region using OpenSky Network data.

## Features
- Live map
- Aircraft type
- Altitude
- Speed
- Place search with suggestions
- Noise score from 1 to 10
- Estimated noise level in dB
- Estimated noise per aircraft card

## Development
- Open `index.html` in a browser.

## Data source
OpenSky Network API
